=== Error Response

모든 에러 Response 기본 포맷입니다.

==== Common

기본 에러 형식입니다.

[source,json]
----
{
  "status": "상태 코드",
  "message": "에러 메세지",
  "errors": [ ]
}
----

==== Details

에러 발생 기점 및 이유를 자세히 설명합니다.

주로 잘못된 입력 검증 시(Input Validation) 나타냅니다.

[source,json]
----
{
  "status": "상태 코드",
  "message": "Error message",
  "errors": [
    {
      "field": "필드 이름",
      "value" : "잘못 입력된 값",
      "reason": "에러 원인"
    }
  ]
}
----
